## Description

This layer adds supports for convenient file manager.

## Install

To use this configuration layer, add it to your `~/.spacevim`.

## Key Bindings

Key Binding                        | Mode   | Description
:---:                              | :---:  | :---:
<kbd>SPC f t</kbd> / <kbd>F4</kbd> | Normal | NERDTree toggle
