## Description

This layer adds support for the elm language.

## Install

To use this configuration layer, add it to your `~/.spacevim`. Like so:

```
Layer 'elm'
```

## Key Bindings

See [ElmCast/elm](https://github.com/ElmCast/elm-vim) for the default key bindings.
